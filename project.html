Database Project - A database system for an art school

I created a database system to keep record of student in an art school. The system also kept data on the courses available, the staff(lecturers and non teaching staff), the exhiitions held, the student projects and the school's affiliates. This system is able to provide information on which lecturer is teaching a certain course, which students are enrolled in a class, which students participated in the exhibition, how much sales the exhibition made and many more.
